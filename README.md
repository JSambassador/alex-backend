# alex-backend

**Program for auth (login, logout, jwt token and refresh), user profile store with Prisma and validation on Typescript and Express.js.**


### Full Project Structure

Here's the full structure of the project:

```
/auth-system
  ├── .env
  ├── jest.config.js
  ├── package.json
  ├── prisma
  │   └── schema.prisma
  ├── src
  │   ├── index.ts
  │   ├── routes
  │   │   ├── auth.ts
  │   │   └── user.ts
  │   └── tests
  │       └── auth.test.ts
  └── tsconfig.json

```
